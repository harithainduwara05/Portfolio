## ✨ Features

- 🎨 **Modern Dark UI** — Glassmorphism design with vibrant gradient accents
- ⌨️ **Typing Animation** — Dynamic role title typewriter effect
- 🌊 **Smooth Animations** — Scroll-based and micro-animations throughout
- 📱 **Fully Responsive** — Optimized for all screen sizes
- 🔢 **Sections** — Hero, About, Skills, Education, Projects, Contact & Footer
- ⬆️ **Back to Top** — Smooth scroll back-to-top button
- ⚡ **Fast Performance** — Built with Vite for lightning-fast load times

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/harithainduwara05/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```
